The DICE Data Collector enables the use of customer data to model successful outcomes based on business benefits.

The Data Collector for DICE is deployed in a customer environment to collect data from vRealize Operations Manager.

The following data is collected:

  • Virtual Machine Object Properties and Metrics
  • Host Machine Object Properties and Metrics
  • Cluster Object Metrics
  • Distributed Virtual Switch Object Metrics
  • Portgroup Object Metrics
  • Datastore Object Metrics

All of this data is dumped into a JSON formatted file that can be uploaded into the DICE tool. {% endblock %}
